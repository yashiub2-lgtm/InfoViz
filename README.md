# 🌌 Skies Across Cultures

A data-driven exploration of how different cultures interpret the night sky through constellations.

## 📊 Overview

This project analyzes 40 sky cultures using constellation data to uncover:

* Shared stars across cultures
* Structural complexity of constellations
* Semantic themes (animals, objects, etc.)
* Cross-cultural similarities and clustering

## 🔍 Key Insights

* Most stars are culture-specific, with few universally shared ones
* Large catalogs (e.g., IAU, Dien) dominate constellation counts
* Cultures can share stars but interpret them differently
* Semantic themes like *animals* and *man-made objects* are most common

## 📈 Visualizations

* Constellations per culture
* Region distribution
* Star universality & degree distribution
* Cultural similarity heatmaps
* Hierarchical clustering
* Semantic profiles (radar charts)

## 🛠️ Tech Stack

* Python (pandas, numpy)
* matplotlib
* scipy (clustering)
* network-based analysis

## 📂 Dataset

* 40 sky cultures
* 500+ constellations
* 1000+ unique stars
